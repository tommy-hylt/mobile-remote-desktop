# React Style Guide

## 1. Components

- **Functional Components**: Use functional components with hooks. Avoid class components.
- **Hooks Rules**: Follow standard React hooks rules.
- **Props**: Use exact interfaces for props validation.

## 2. JSX

- **Simplicity**: Keep JSX structure clean and readable. Extract complex render logic into sub-components.
- **Events**: Use inline anonymous functions for simple event handlers to avoid clutter.
